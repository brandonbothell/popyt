Popyt

A very easy to use Youtube Data API wrapper.

Popyt Tutorial - 5min ⏱️

Easy to Use

Popyt was designed from the ground up to be quick to install and use to download from and upload to the YouTube Data API intuitively.

Focus on What Matters

Popyt lets you focus on the data you want to retrieve, while it works behind the scenes to parse the data from the YouTube API into a easy-to-use, well-documented format.

Save Your Quota

Don't worry about making multiple calls to functions for data. Popyt will automatically cache data for a specifiable amount of time.
